# ✅ Быстрая проверка исправления прокрутки select

## Что было исправлено
В мобильной версии dropdown с выбором категории запчастей теперь корректно прокручивается.

## Как проверить

### На мобильном устройстве:

1. Откройте приложение на телефоне:
   - http://192.168.1.127:8000/mechanic
   - или через ngrok/туннель

2. Нажмите на поле **"Категория запчастей"**

3. Попробуйте прокрутить список категорий:
   - Лампочки ✓
   - жидкости\масла\ ✓
   - типуль ✓
   - типуль\кузов ✓
   - тормоза ✓
   - и т.д.

### Что должно работать:
✅ Список прокручивается плавно
✅ Можно выбрать любую категорию
✅ Не требуется зум экрана
✅ Нативный вид для iOS/Android

### Файлы с изменениями:
- ✅ `templates/mechanic.html` - добавлены стили для select
- ✅ `static/css/mobile-responsive.css` - глобальные мобильные стили
- ✅ `templates/mechanic/order_form.html` - форма заказа механика

## Техническое решение
Используется нативный системный select с поддержкой прокрутки через:
- `appearance: menulist-button` 
- `font-size: 16px` (предотвращает зум на iOS)
- `-webkit-overflow-scrolling: touch` (плавная прокрутка)
