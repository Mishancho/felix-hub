## 🔍 Как проверить

### 1. Chrome DevTools:
```
Cmd+Opt+I  →  Cmd+Shift+M  →  Выбрать устройство
```

### 2. Тестовые устройства:
- iPhone SE (375px) - минимум
- iPhone 14 Pro (393px) - стандарт
- Pixel 7 (412px) - Android
- iPad (768px) - планшет

### 3. Быстрая проверка каждой страницы:
```bash
✓ Нет →← скролла
✓ Кнопки >= 44px
✓ Текст читаем
✓ Формы удобны
✓ Модалки работают
```

## 💡 Полезные классы

### Показать только на мобильных:
```html
<div class="mobile-only">
    Только на мобильных (<768px)
</div>
```

### Скрыть на мобильных:
```html
<div class="desktop-only">
    Только на десктопе (>768px)
</div>
```

### Скрыть колонку таблицы:
```html
<th class="hide-mobile">Email</th>
<td class="hide-mobile">user@example.com</td>
```
