# Исправление отступов контента ✅

## Проблемы которые были исправлены

### 1. ❌ Страница "Мои заказы" не открывалась
**Причина:** Синтаксическая ошибка в файле `templates/mechanic/orders.html`
- В строке 2 был поврежден тег `<html>` - случайно остался CSS комментарий

**Решение:**
```html
<!-- БЫЛО (с ошибкой): -->
<html lang="{{ g.lo        /* Стили навигации перенесены в fixed-header-nav.css */

<!-- СТАЛО: -->
<html lang="{{ g.locale }}" dir="{{ 'rtl' if g.locale == 'he' else 'ltr' }}">
```

### 2. ❌ Контент уходил под фиксированную шапку и меню
**Причина:** Недостаточный отступ сверху для body
- Шапка имеет высоту 85px (desktop)
- Навигационное меню занимает еще ~85px с отступами
- Но `body padding-top` был только 155px

**Решение:** Увеличен отступ в `static/css/fixed-header-nav.css`

## Внесенные изменения

### Файл: `static/css/fixed-header-nav.css`

#### Desktop версия (>768px):
```css
body {
    padding-top: 170px; /* Шапка 85px + меню ~85px */
}

.container,
.main-content {
    padding-top: 0;
    margin-top: 0;
}
```

#### Tablet версия (≤768px):
```css
body {
    padding-top: 145px; /* Шапка 70px + меню ~75px */
}
```

#### Mobile версия (≤480px):
```css
body {
    padding-top: 125px; /* Шапка 60px + меню ~65px */
}
```
